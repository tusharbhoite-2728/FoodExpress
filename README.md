# FoodExpress
FoodExpress is a multi-vendor food delivery web application built using JSP, Servlets, and JDBC (without Maven). It offers role-based dashboards for customers, vendors, delivery personnel, and admins, enabling seamless food ordering, order management, and delivery tracking.

Welcome to FoodExpress, a dynamic and user-friendly multi-vendor food delivery web application built using JSP, Servlets, and JDBC. Designed with simplicity and scalability in mind, FoodExpress connects hungry customers with their favorite local restaurants, streamlining the entire ordering and delivery experience.
